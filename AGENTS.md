# Project Instructions

This is a frontend-only company website for Agrani Technologies & Services Limited.

Use:
- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- lucide-react

Do not add:
- Backend
- Database
- Authentication
- CMS
- Admin panel

The contact form should be frontend-only for now. Show a success message on submit and include a mailto fallback.

Always keep the website responsive, modern, animated, and production-quality.
